# Netflix Power BI Dashboard

## 📊 Project Overview
This project is an interactive Netflix Dashboard built using Power BI. It analyzes Netflix movies and TV shows using visualizations and KPIs.

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV

## 📈 Dashboard Features
- Total Movies & TV Shows
- Content by Country
- Ratings Distribution
- Genre Analysis
- Release Year Trends
- Interactive Filters

## 📂 Dataset
Netflix Titles Dataset (included in this repository)
